The JSR-107 specification

by the JSR-107 expert group

JSR-107 is a specification for a temporary caching API for Java, as defined by the Java Community Process (JCP). More information is available on the JCP's specification page for JSR-107.

The specification itself, as well as the reference implementation and TCK, is hosted on GitHub. For more details, visit the project's GitHub repository.
